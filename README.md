# Estudos Java

Repositório dedicado ao aprendizado da linguagem Java e lógica de programação orientada a objetos.

## 🏦 Projeto: Sistema Bancário
O primeiro projeto foca em **Encapsulamento** e **Separação de Responsabilidades**.

### Conceitos aplicados:
* **Classes e Objetos**: Estrutura principal do sistema.
* **Encapsulamento**: Atributos `private` e métodos `public`.
* **Construtores**: Inicialização segura de dados.
* **Interface de Usuário**: Uso da biblioteca `Swing` (JOptionPane).
